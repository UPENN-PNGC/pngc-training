# Using AlphaGenome - Introductory Practical Session

## Description

AlphaGenome offers multimodal predictions, encompassing diverse functional outputs such as gene expression, splicing patterns, chromatin features, and contact maps. The model analyzes DNA sequences of up to 1 million base pairs in length and can deliver predictions at single base-pair resolution for most outputs. This seminar introduces basic inputs and outputs, research uses, and practical steps for getting started with the freely available Preview.
