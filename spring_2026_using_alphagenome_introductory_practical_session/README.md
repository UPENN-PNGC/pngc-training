# Using AlphaGenome - Introductory Practical Session
    ## Description
    AlphaGenome provides regulatory impact scoring, variant effect prediction, gene and tissue context, and expression signals.  This seminar introduces basic inputs and outputs, research uses, and practical steps for getting started with the freely available Preview.
